== Description ==

The dofollow plugin for WordPress lets you remove the evil nofollow attribute from your comments. Because [nofollow will not work]


= Help Me! =

If you need assistance, consider using (http://www.wordpresssmarts.com).


== Installation ==

  
1. Unzip files to your local Hard Disk.
2. Upload "wp-dofollow.php" file to your website plugin folder (http://yourwebsite.com/wp-content/plugin)
3. Login to web site admin section & active plugin
4. Set configuration (Settings > Wp-Dofollow)

Unzip files to the main "plugin" root folder.
